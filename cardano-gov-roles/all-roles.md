# Cardano Ecosystem Roles
A overview on current ecosystem roles within the Cardano ecosystem

|Category |Description |Tier|
|---------|------------|----|
|Explorer|Beginning the journey, discovering the ecosystem|A|
|Builder|Actively contributing, shaping the environment|B|
|Architect|Designing structures and guiding the ecosystemâ€™s evolution|C|
|Guardian|Safeguarding principles, ensuring sustainability and integrity|D|

------

Note*
Community members can engage across multiple roles at the same time which is usually encouraged.
